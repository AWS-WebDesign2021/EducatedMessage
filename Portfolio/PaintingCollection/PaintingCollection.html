This was our first webpage with multiple webpages and relative links. This webpage had quite a few requirements, which included 4 web pages that could relatively link to each other and to a homepage, use of a table, 2 different divs and styles, images and other media, citations and comments. This project was my first time using tables and trying to figure out how to make a table was the largest problem I had encountered at that point. My completed table wasn’t what I had initially wanted, but I was really proud of myself for using different resources and problem-solving skills to figure it out. We used hyperlinks for the first time, which are pretty simple, but I was really excited to learn how to do it, because they add a lot more of a professional look to web pages. Our comments also started to get more substantial, even if they weren’t used for all the right places or expressed everything they needed to.
